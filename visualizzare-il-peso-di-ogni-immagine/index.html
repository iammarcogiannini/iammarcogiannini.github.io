Image KB Viewer - Un estensione per leggere il peso delle immagini

Una volta, non ricordo bene il motivo, mi son trovato a dover leggere il peso (in KB) di molte immagini. Ho subito cercato qualche soluzione per visualizzare tutte le dimensioni in modo automatico senza dover scaricare ogni immagine o controllarle dalla console di Chrome.

Estensioni online non ne ho trovate che facessero esattamente quello che volevo quindi ho pensato di farmela. Come sempre parte la sfida con me stesso.

In realtà qui la sfida è stata abbastanza semplice. Mi ero da poco allenato su fetch.

The Fetch API provides a JavaScript interface for accessing and manipulating parts of the HTTP pipeline, such as requests and responses. It also provides a global fetch() method that provides an easy, logical way to fetch resources asynchronously across the network.

Con fetch abbiamo a disposizione svariati metodi per estrarre delle response dal body. Tra questi abbiamo `.blob()`. Il metodo `.blob()` ritorna una blob promise. Da questo campo è possibile leggere la dimensione in byte di un file.

È stato quindi molto semplice convertire i byte in Kbyte e poi mostrarli a schermo manipolando il dom con JS.

Qui potete trovare la repository di Github: https://github.com/gianemi2/image-kb-viewer per essere installata dovete attivare la modalità sviluppatore di Chrome.
